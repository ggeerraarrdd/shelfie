Shelfie! is a management system for personal home libraries. It is written in Python using the Tkinter module with a bit of SQL thrown in.

You can browse, add, update and delete records saved a SQLite database.

It is a project by Gerard Bul-lalayao for IS 411: Intro to Programming for Business Application (Autumn 2022).
